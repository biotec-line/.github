# biotec-line

**Bioinformatics software for genotype conversion, VCF processing, and variant-centric desktop workflows.**

biotec-line focuses on practical tools for working with genetic variant data, from direct-to-consumer DNA raw files to annotated VCF-based downstream analysis.

## Public Projects

| Project | Description |
|---------|-------------|
| [VFDistiller](https://github.com/biotec-line/VFDistiller) | Desktop-oriented variant processing pipeline with conversion, filtering, and multi-source annotation |
| [genotype-to-vcf](https://github.com/biotec-line/genotype-to-vcf) | Convert raw DNA exports from services like 23andMe and MyHeritage to VCF 4.2 |

## Related Research

biotec-line is closely connected to [research-line](https://github.com/research-line), where the open-access research and publication side of the ecosystem lives.
