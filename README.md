# biotec-line .github

<!-- public-index-last-checked: 2026-06-21 -->

Organization profile and community health files for [biotec-line](https://github.com/biotec-line).

This repository controls the public organization overview shown on GitHub and the shared contribution, security, issue, and pull request templates used across the biotec-line bioinformatics projects.

The public profile README is also the maintained repository directory for the organization: it lists all public biotec-line repositories and deliberately excludes private, internal, or unreleased work. The current public index was verified against GitHub on 2026-06-21.

## Public Repository Directory

| Repository | Public role | Search/discovery focus |
|---|---|---|
| [.github](https://github.com/biotec-line/.github) | Organization landing page, shared community-health files, and `llms.txt` crawler context | biotec-line organization profile, local-first bioinformatics tools |
| [VFDistiller](https://github.com/biotec-line/VFDistiller) | Local-first VCF/gVCF annotation, filtering, validation, and export desktop app for research use only | VCF annotation desktop app, gVCF filtering GUI, gnomAD, FASTA, variant workflows |
| [genotype-to-vcf](https://github.com/biotec-line/genotype-to-vcf) | DTC DNA raw-data to VCF 4.2 converter with GUI and headless CLI paths | 23andMe to VCF, MyHeritage VCF conversion, GRCh37/GRCh38, dbSNP, FASTA REF lookup |
